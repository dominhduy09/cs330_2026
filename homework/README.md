# CS330 Homework Assignments

This directory contains all homework assignments for CS330.

## Assignment Structure

Each homework will be in its own subdirectory with the following structure:

```
hwX/
├── README.md           # Assignment description and instructions
├── starter_code/       # Starter code and templates
├── data/              # Dataset or data loading scripts
└── submission/        # Your submission files
```

## Submission Guidelines

### General Instructions

1. Complete all problems in the assignment
2. Include all required code files
3. Write clear comments explaining your implementation
4. Include a brief report (when specified) explaining your approach and results
5. Test your code before submission

### Code Requirements

- Use Python 3.8+
- Follow PEP 8 style guidelines
- Include docstrings for functions
- Make your code reproducible (set random seeds where applicable)
- Include a requirements.txt if using additional packages

### What to Submit

For each assignment, submit:
- All Python files (.py) or Jupyter notebooks (.ipynb)
- A written report (PDF format) when required
- Any additional files specified in the assignment

### File Naming Convention

```
hwX_studentID_lastname.zip
```

For example:
```
hw1_123456_smith.zip
```

## Grading Criteria

Assignments will be graded on:
- **Correctness (60%)**: Does your code produce correct results?
- **Code Quality (20%)**: Is your code well-organized, commented, and readable?
- **Analysis (15%)**: Do you provide insightful analysis of results?
- **Efficiency (5%)**: Is your implementation reasonably efficient?

## Late Policy

- Assignments submitted up to 24 hours late: 10% penalty
- Assignments submitted 24-48 hours late: 25% penalty
- Assignments submitted more than 48 hours late: Not accepted

Each student has 3 "late days" to use throughout the course without penalty.

## Collaboration Policy

- You may discuss approaches and concepts with classmates
- You must write your own code independently
- You must acknowledge any collaborators in your submission
- Copying code from others or online sources is prohibited
