# CS330 Course Syllabus (2026)

## Course Description

CS330 explores fundamental topics in deep multi-task learning and meta-learning. Students will learn about algorithms that enable models to learn new tasks quickly from limited data and adapt to new domains.

## Learning Objectives

By the end of this course, students will be able to:
- Understand the principles of multi-task learning and transfer learning
- Implement meta-learning algorithms
- Design and train models for few-shot learning scenarios
- Apply meta-learning techniques to real-world problems
- Critically evaluate research papers in the field

## Topics Covered

### Module 1: Introduction to Multi-Task Learning
- Basics of multi-task learning
- Transfer learning fundamentals
- Task relationships and negative transfer

### Module 2: Meta-Learning Fundamentals
- What is meta-learning?
- Problem formulation
- Common benchmarks and datasets

### Module 3: Optimization-Based Meta-Learning
- Model-Agnostic Meta-Learning (MAML)
- First-order approximations
- Task conditioning

### Module 4: Metric Learning and Few-Shot Classification
- Prototypical networks
- Matching networks
- Relation networks

### Module 5: Advanced Topics
- Contextual policies
- Black-box meta-learning
- Meta-reinforcement learning
- Recent advances in the field
